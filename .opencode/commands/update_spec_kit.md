description: VISION — update_spec_kit (actualizar templates desde github/spec-kit)
---

Actualiza los templates del Spec Kit desde el repositorio upstream.

## Instrucciones

1. Verifica que git está disponible en el sistema
2. Ejecuta: `node scripts/update-spec-kit.js`
3. El script clona/actualiza `spec-kit/templates/` desde el repositorio configurado
4. Informa qué templates fueron actualizados/añadidos

**Nota:** Requiere conexión a internet y git configurado.
