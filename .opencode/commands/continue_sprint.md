---
description: VISION — continue_sprint (retomar sprint activo incompleto)
---

Retoma el sprint activo con tareas pendientes.

## Instrucciones

1. Lee `planning/sprints/sprint-0N/sprint-state.json` para identificar el sprint activo.
2. Lee `tasks.md`, `stories.md` y `qa-plan.md` del sprint activo.
3. Continúa implementando las tareas pendientes (no marcadas como `done`).
4. Marca `done` en `tasks.md` al completar cada tarea.

**Siguiente acción:** Cuando todas las tareas estén `done`, usa `start_sprint --sprint N+1` para iniciar el siguiente sprint.
