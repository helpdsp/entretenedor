---
description: VISION — update_agency_agents (actualizar definiciones de agentes)
---

Actualiza las definiciones de agentes desde el repositorio upstream.

## Instrucciones

1. Verifica que git está disponible en el sistema
2. Ejecuta: `node scripts/update-agency-agents.js`
3. El script clona/actualiza `agency-agents/` desde el repositorio configurado
4. Informa qué agentes fueron actualizados/añadidos

**Nota:** Requiere conexión a internet y git configurado.
