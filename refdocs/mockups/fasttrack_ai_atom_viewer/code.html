Sintaxis básica

El Hook useEffect le permite a React conectarse con sistemas externos. Esto incluye manipular el DOM, peticiones API o suscripciones.

javascript content_copy 
useEffect(() => {
  // Tu lógica de efecto aquí
  console.log('El componente se montó');
}, [dependencias]);
1

Función de configuración

Define el código que quieres ejecutar después de que el componente se renderice.

2

Array de dependencias

Controla cuándo debe volver a ejecutarse el efecto basándose en cambios de estado.

lightbulb Ejemplo práctico

Imagina que quieres actualizar el título de la página cada vez que un contador cambie. Usarías useEffect para sincronizar el estado de React con la API del navegador.
