Sintaxis básica

El hook useEffect permite ejecutar efectos secundarios en componentes funcionales. Es la herramienta principal de React para manejar operaciones que ocurren fuera del flujo de renderizado normal.

info ¿Cómo funciona?

Piensa en useEffect como un observador. React ejecutará tu función de efecto después de que el renderizado se haya aplicado a la pantalla. Esto asegura que no bloquees la interactividad del usuario con tareas pesadas o asíncronas.

BasicEffect.js content_copy 
useEffect(() => {
  // 1. El código de tu efecto aquí
  console.log('Componente montado');

  // Opcional: Función de limpieza
  return () => {
    console.log('Limpiando efecto');
  };
}, [dependency]); // 2. El array de dependencias
1

Setup (Función de Efecto)

Es la función que contiene la lógica. Se ejecuta por defecto después del primer renderizado y tras cada actualización, a menos que se especifique lo contrario.

2

Dependency Array

Un arreglo de valores. React solo volverá a ejecutar el efecto si alguno de estos valores cambia entre renderizados. Un array vacío [] significa que solo se ejecuta una vez al montar.
