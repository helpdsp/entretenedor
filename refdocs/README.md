# Refdocs

Coloca aqui documentos de referencia del proyecto (objetivos, requerimientos, contexto de negocio, etc.).

Pueden ser **varios archivos** y distintos formatos (texto, exportes de PDF/DOCX, transcripciones de audio, notas derivadas de imagenes). Antes de `generate_spec_kit`, conviene consolidar un **PRD de producto** unico (por ejemplo `spec-kit/input/product-prd.md` o un `*PRD*.md` en esta carpeta).

Regla minima para `generate_brief`:

- Debe existir al menos 1 archivo en esta carpeta.
